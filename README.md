# FMI Stochastics course archive

Static GitHub Pages package for `https://fmi-stochastics.github.io/`.

## Upload

1. Extract this ZIP locally.
2. Upload the extracted contents to the root of the repository `fmi-stochastics/fmi-stochastics.github.io`.
3. Commit the changes.
4. Keep GitHub Pages set to `Deploy from a branch`, branch `main`, folder `/root`.

The site is pure static HTML/CSS. No build step is required.

## File naming convention

All public files were copied into `files/` and renamed using lowercase Latin filenames, hyphens, and stable programme/course prefixes. The original names are recorded in `MANIFEST.csv`.

## Privacy

Do not upload grades, student submissions, scanned exams with visible student names, or unreleased exam drafts.
