# Вероятности СУ — MkDocs сайт

Това е изходният код на сайта. Публичният сайт се генерира автоматично от GitHub Actions.

## Обновяване

1. Сложи новите PDF файлове в `docs/files/...`.
2. Редактирай съответната Markdown страница в `docs/prob-stat/`.
3. Commit.
4. Push.

## Локална проверка по желание

```bash
pip install -r requirements.txt
mkdocs serve
```
